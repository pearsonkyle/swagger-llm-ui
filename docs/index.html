AI-Enhanced API Documentation

Load any OpenAPI schema and explore it with Chat, Workflow, and Agent assistance — powered by your local LLM.

Enter a direct openapi.json URL, or paste a /docs page URL and DocBuddy will auto-detect the schema.

Python Plugin

Replace your FastAPI /docs page with DocBuddy — get AI chat, workflows, and agent tools directly alongside your API explorer.

from fastapi import FastAPI from docbuddy import setup_docs app = FastAPI() setup_docs(app) # replaces default /docs
